블로그 포스트는 **Markdown**으로 작성합니다. 이 포스트에서는 자주 사용하는 Markdown 문법을 정리합니다.

## 제목 (Headings)

```markdown
## h2 제목
### h3 제목
#### h4 제목
```

## 텍스트 스타일링

| 스타일 | 문법 | 결과 |
| ------ | ---- | ---- |
| 굵게 | `**텍스트**` | **텍스트** |
| 기울임 | `*텍스트*` | *텍스트* |
| 취소선 | `~~텍스트~~` | ~~텍스트~~ |

## 코드

인라인 코드는 `` `백틱` ``으로 감싸면 됩니다.

코드 블록은 세 개의 백틱으로 감싸고 언어를 지정합니다:

```javascript
function greet(name) {
  console.log(`안녕하세요, ${name}님!`);
}

greet('World');
```

## 리스트

순서가 없는 리스트:
- 첫 번째 항목
- 두 번째 항목
  - 중첩된 항목
- 세 번째 항목

순서가 있는 리스트:
1. 첫 번째 단계
2. 두 번째 단계
3. 세 번째 단계

## 인용문

> 좋은 코드는 최고의 문서화다. 코드에 주석을 추가하려 할 때, 스스로에게 물어보라:
> "이 코드를 어떻게 개선하면 주석이 필요없어질까?"
> — Steve McConnell
